
# RiddleSwap Privacy Policy

Last updated: 2025-05-28

## Information We Collect
- Wallet addresses (for transaction display)
- Transaction history (for user interface)
- App usage analytics (anonymous)

## Information We Don't Collect
- Private keys or seed phrases
- Personal identification information
- Financial information beyond public blockchain data

## Third-Party Services
- DexScreener API (for token pricing data)
- XRPL Network (for blockchain interactions)
- Xaman Wallet (for transaction signing)

## Data Storage
- Transaction history stored locally on device
- No personal data transmitted to our servers
- All wallet interactions handled by Xaman

## Contact
For questions about this policy, contact: support@riddlechain.io

This policy complies with Google Play Store requirements.
